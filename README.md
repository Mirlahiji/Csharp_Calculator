## ABOUT windows form calculator: 
This is an __Open-source__ calculator build with visual studio form for C# languge which is made for Windows and can do some basic calculation with numbers.

### FEATURES :
- **Addition**
- **Subtraction**
- **Division**
- **Multiplication**
- **Power**
- **Square Root**
- **Natural Logarithm**
- **10 Base Logarithm**
